Demo: https://vladlent.github.io/dynamic-table/

About

Main purpose of this app is to take JSON as an input and generate table from it. It's built with Angular 8 and NgRx 8.

Features:

  1. Can sort the table by clicking on column header. First click sorts by ASC second by DESC.
  2. Any table column (except header) can be edited by clicking on it.
  3. Search with highlighted query in results.
  4. Can read JSON by either providing a link to it or by uploading a file.
  5. Drag'n'drop.
  6. Pagination with ability to select amount of entries per page.
  7. 100% unit-test coverage.

Plans:

  • Support for nested objects.
  • Ability to download edited JSON.
  • Error notifications.
